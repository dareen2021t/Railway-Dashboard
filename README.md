🚆Railway Analytics Dashboard
🧭 Overview

This project presents an interactive Power BI dashboard analyzing railway performance, including revenue, journey status, delays, and cancellations across different routes, cities, and time periods.

 Key Features

 Revenue Analysis by Ticket Type, Class, and Payment Method
 Journey Status Breakdown (Completed, Delayed, Cancelled)
 Delay & Cancellation Insights
 Top & Bottom Performing Cities
 Time-Based Trends (Monthly Performance)
 Railcard & Passenger Behavior Analysis
